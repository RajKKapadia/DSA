# DSA Practice

This repository contains Python solutions for data structures and algorithms practice problems. Each problem lives in its own folder with a `main.py` file that includes:

- test cases for the problem
- one or more solution approaches
- a runnable test loop that prints consistent pass/fail results

## Problems

- `two-sum`
- `two-sum-sorted-array`
- `container-with-most-water`
- `move-zeroes`
- `best-price-to-buy-sell-stock`
- `remove-duplicated-sorted-array`
- `three-sum-problem`

## Running Tests

Run any problem directly from its folder:

```bash
python3 main.py
```

Or run one from the repository root:

```bash
python3 two-sum/main.py
```

Each script prints one line per test case and a final summary:

```text
Test 1: PASSED - case 1
Summary: 15/15 tests passed
```
